# Tribute Page: Ada Lovelace

A responsive tribute page dedicated to **Ada Lovelace**, the 19th-century mathematician widely regarded as the world's first computer programmer. 

> "That brain of mine is something more than merely mortal; as time will show." — Ada Lovelace

---

##  Project Overview

This project is a static website built to honor the legacy of Ada Lovelace. It highlights her collaboration with Charles Babbage and her visionary understanding of the Analytical Engine's potential beyond mere calculation.

### Features
* **Semantic HTML5**: Structured using accessible elements like `<main>`, `<figure>`, and `<section>`.
* **Historical Timeline**: A summary of her major contributions between 1842 and 1843.
* **Responsive Layout**: Designed to be readable across different device screen sizes.
* **Direct Documentation**: Includes a functional link to her full biography for further learning.

##  Built With

* **HTML5** - For the page structure and content.
* **CSS3** - For custom styling and layout (linked via `styles.css`).
